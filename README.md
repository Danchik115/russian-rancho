# Русское Ранчо — сайт

Сайт работает на **Django**.

## Страницы
- Главная — форматы, популярные программы, ближайшие события, преимущества, контакты/заявка
- Для семей, для детей, для пар, для групп
- Афиша, цены, о ферме, галерея
- Тепло и уют для вас (домики и беседки)

## Как запустить

```bash
cd russian-rancho
pip install -r requirements.txt
python manage.py runserver
```

Открыть в браузере: **http://127.0.0.1:8000/**

Страницы: `/`, `/about/`, `/families/`, `/kids/`, `/couples/`, `/groups/`, `/events/`, `/prices/`, `/gallery/`, `/nedvizhimost/`.

## Форма заявки и отправка в Telegram

Заявка с главной страницы (блок «Контакты и запись») может отправляться в Telegram.

### Как включить отправку в Telegram

1. **Создайте бота в Telegram**
   - Напишите [@BotFather](https://t.me/BotFather), команда `/newbot`.
   - Сохраните выданный **токен** (например, `123456:ABC-DEF...`).

2. **Узнайте свой chat_id**
   - Напишите своему боту любое сообщение.
   - Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`.
   - В ответе найдите `"chat":{"id": 123456789}` — это ваш **chat_id**.

3. **Автоотправка заявок в Telegram**
   - Для Django-версии сайта задайте переменные окружения:
     - `TELEGRAM_BOT_TOKEN` — токен бота
     - `TELEGRAM_CHAT_ID` — ваш chat_id (число)
     - `TELEGRAM_API_BASE` — адрес Telegram Bot API (по умолчанию `https://api.telegram.org`)
   - Проще всего: скопируйте `.env.example` в `.env` и заполните значения.
   - После этого формы будут отправляться через `POST /api/telegram` автоматически.

Пример:
```bash
cp .env.example .env
# затем отредактируйте .env
python manage.py runserver
```

Если в вашей сети блокируется `api.telegram.org`, используйте VPN/прокси или вынесите сервер в сеть без блокировки.

После этого при отправке формы заявка будет приходить вам в Telegram автоматически. Если API недоступен, покажется текст заявки — его можно скопировать и отправить вручную.

## Деплой на Render (вариант 2)

В репозитории уже есть `render.yaml`, поэтому можно развернуть в пару кликов:

1. Залей изменения в GitHub.
2. В Render: **New +** → **Blueprint** → выбери репозиторий.
3. При создании сервиса задай секреты:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
4. После деплоя открой сайт по URL Render и отправь тестовую заявку.

Если домен сервиса отличается от `russian-rancho.onrender.com`, обнови `DJANGO_ALLOWED_HOSTS` в Render (через запятую можно несколько хостов).
