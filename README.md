# Русское Ранчо — сайт (статический)

Это статический сайт (HTML/CSS/JS) по концепции «Русское Ранчо»: главная страница продаёт **конкретные форматы посещения** (семьи/дети/пары/группы/мероприятия) и ведёт на отдельные страницы.

## Страницы
- `index.html` — главная (форматы, популярные программы, ближайшие события, преимущества, контакты/заявка)
- `families.html` — для семей
- `kids.html` — для детей
- `couples.html` — для пар
- `groups.html` — для групп
- `events.html` — афиша
- `prices.html` — цены
- `about.html` — о ферме
- `gallery.html` — галерея

## Как запустить
Проще всего открыть `index.html` двойным кликом.

Если хочешь локальный сервер (чтобы всё выглядело так же, как на хостинге):

```bash
cd russian-rancho
python3 -m http.server 8000
```

Открыть в браузере: `http://localhost:8000`

## Форма заявки и отправка в Telegram

Заявка с главной страницы (блок «Контакты и запись») может отправляться в Telegram.

### Как включить отправку в Telegram

1. **Создайте бота в Telegram**
   - Напишите [@BotFather](https://t.me/BotFather), команда `/newbot`.
   - Сохраните выданный **токен** (например, `123456:ABC-DEF...`).

2. **Узнайте свой chat_id**
   - Напишите своему боту любое сообщение.
   - Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`.
   - В ответе найдите `"chat":{"id": 123456789}` — это ваш **chat_id**.

3. **Разверните сайт на Vercel** (чтобы работал API)
   - Зайдите на [vercel.com](https://vercel.com), подключите репозиторий GitHub.
   - В настройках проекта → **Environment Variables** добавьте:
     - `TELEGRAM_BOT_TOKEN` — токен бота
     - `TELEGRAM_CHAT_ID` — ваш chat_id (число)
   - Деплой. Сайт будет по адресу вида `https://ваш-проект.vercel.app`.

4. **Если сайт открыт с GitHub Pages**, а API на Vercel  
   В `index.html` перед подключением `script.js` добавьте (подставьте свой URL Vercel):
   ```html
   <script>window.FORM_API_URL = "https://ваш-проект.vercel.app";</script>
   ```

После этого при отправке формы заявка будет приходить вам в Telegram. Если API недоступен (например, открыли сайт локально), покажется текст заявки — его можно скопировать и отправить вручную.
